이 리포지토리는 **GitHub Actions**를 활용하여 매일 정보보안기사 핵심 이론을 자동으로 학습하고 기록하는 프로젝트입니다.

## 🚀 프로젝트 개요
* **자동 생성**: 매일 오전 9시, Python 스크립트가 보안 지식 베이스에서 랜덤 주제를 선정합니다.
* **블로그 연동**: 생성된 학습 내용은 `_posts/` 디렉토리에 저장되어 Jekyll/Hugo 블로그에 자동 포스팅됩니다.
* **학습 현황판**: 이 README 파일에서 최신 학습 내역을 바로 확인할 수 있습니다.

## 📚 학습 과목 (Knowledge Base)
시스템 보안, 네트워크 보안, 어플리케이션 보안, 정보보안 일반, 관리 및 법규 등 5대 영역의 방대한 데이터를 포함하고 있습니다.
